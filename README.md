# AGID PASS

O **AGID PASS** é uma plataforma online que centraliza **planos e serviços de beleza** em um único lugar, facilitando o acesso a salões, profissionais e tratamentos. Com ele, os usuários podem explorar diferentes serviços, contratar planos personalizados e gerenciar suas reservas de forma prática e segura.

## Protótipo
https://www.figma.com/proto/SYkA3S0pTJPO7oxxOfW5Wo/AGID-SITE?node-id=9-524&starting-point-node-id=9%3A524

## Funcionalidades
- Consulta de profissionais e salões disponíveis.
- Contratação de planos de beleza personalizados.
- Agendamento de serviços com poucos cliques.
- Interface intuitiva e responsiva para desktop e mobile.
- Histórico de serviços e gerenciamento de reservas.

## Tecnologias Utilizadas
- **Frontend:** Angular, HTML, CSS, JavaScript  
- **Backend:** Node.js
- **Banco de Dados:** MySQL  
- **Controle de Versão:** GitHub
